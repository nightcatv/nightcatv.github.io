Flask - Email

2019-05-05
Python / Flask

Use Flask-Mail

  • Python標準函式庫含smtlib套件寄送email，但Flask-Mail包裝了smtlib並將其與Flask做整合

    • 安裝Flask-Mail
    • 會連接 簡易郵件傳輸通訊協定(SMTP) 伺服器 => 127.0.0.1:25
    • 設置SMTP伺服器的組態鍵清單
    鍵 預設值 說明
    MAIL_SERVER localhost email伺服器的主機名稱 / IP位址
    MAIL_PORT 25 email伺服器的連接埠
    MAIL_USE_TLS False 啟用傳輸層安全性(TLS)
    MAIL_USE_SSL False 啟用安全通訊端層(SSL)
    MAIL_USERNAME None 郵件帳號的使用者名稱
    MAIL_PASSWORD None 郵件帳號的密碼

Begin with Flask

2019-02-24
Python / Flask

What is Flask?

  • 使用Python寫的輕量級Web應用框架。
  • 通常被稱為 “microframework(微框架)”，原因：
    • 使用簡單的核心
    • 用extension增加其他功能
  • 沒有預設的資料庫、表單驗證工具 => 保留了擴增的彈性
  • Flask是基於下列工具所開發出來的：
    • WSGI
      • Web Server Gate Interface
      • 為Python網頁應用程式開發的標準，主要為介於網路應用程式與伺服器之間的規範
    • Werkzeug
      • WSGI的工具
      • 允許網頁框架以此為基礎來建立
    • Jinja2
      • Python的模板引擎 (Templating Engine)

Web-crawler in Python

2018-09-07
Python

What is Web-crawler?

  • 一種機器人，自動爬取目標網頁並依需求蒐集目標資料
  • 步驟：
    1. 取得指定網域下的HTML資料
    2. 解析這些資料以取得目標資料
    3. Loop
  • 套件簡介：
    • requests => 用來對目標網頁的server發出request，底層為urllib
    • BeautifulSoup => 用來解析html，底層為re
    • pandas => 爬取表格
    • selenium => 網頁測試工具，用來應付較麻煩的JavaScript
    • re => 正則表達式，用以取技術性較高的文字段落
